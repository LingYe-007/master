# 联邦论文推荐系统 - 前端

与论文第五章「基于联邦异构图模型的推荐系统设计与实现」对应的 Web 前端，技术栈：**React + TypeScript + Ant Design + ECharts**。

## 功能

- **登录 / 注册**：联邦身份认证入口（无后端时可使用演示账号）
- **学术推荐**：论文列表、结构优先/属性优先策略标识、推荐理由（元路径可解释性）
- **个人设置**：研究兴趣、所属机构、研究方向（个人画像，用于冷启动）
- **训练监控**（管理员）：当前轮次、客户端数、Loss/AUC 曲线、客户端在线状态

## 本地运行

```bash
cd fedrec-frontend
npm install
npm run dev
```

浏览器访问 http://localhost:3000 。

## 无后端演示

未启动后端时：

- **登录**：使用 `demo` / `demo` 以管理员身份进入（可访问「训练监控」）；或 `user` / `user` 以普通用户进入。
- 推荐列表与监控数据使用前端 Mock，与论文描述一致。

## 对接后端

1. 配置环境变量：复制 `.env.example` 为 `.env`，设置 `VITE_API_BASE_URL` 为后端 API 根路径。
2. 开发时可用 Vite 代理：在 `vite.config.ts` 中已配置 `/api` -> `http://localhost:5000`，后端需提供 `/api/auth/login`、`/api/user/me`、`/api/recommend`、`/api/admin/training` 等接口。
