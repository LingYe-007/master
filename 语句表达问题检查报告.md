# 论文语句表达问题检查报告

## 一、语法错误与用词不当

### 1.1 第一章（chap1_intro.tex）

**问题1：口语化表达**
- **位置**：第34行
- **原文**：`联邦学习通常需要数几百轮的迭代才能收敛`
- **问题**："数几百轮"是口语化表达，不符合学术写作规范
- **建议修改为**：`联邦学习通常需要数百轮的迭代才能收敛` 或 `联邦学习通常需要数百至数千轮的迭代才能收敛`

**问题2：表达不够学术化**
- **位置**：第39行
- **原文**：`因此对于该课题的研究极具意义`
- **问题**："极具意义"表达不够正式
- **建议修改为**：`因此该课题的研究具有重要的理论价值和实践意义` 或 `因此深入研究该课题具有重要意义`

**问题3：术语不一致**
- **位置**：第57行
- **原文**：`确立了轻量级图协同过滤的 SOTA 基准`
- **问题**：根据之前的修改要求，应统一使用"基准"而非"SOTA基准"
- **建议修改为**：`确立了轻量级图协同过滤的基准`

**问题4：句子不完整**
- **位置**：第89行
- **原文**：`对齐；FedDCSR \cite{zhang2024feddcsr}。`
- **问题**：只有引用，没有对FedDCSR方法的描述，句子不完整
- **建议修改为**：补充FedDCSR方法的简要描述，例如：`对齐；FedDCSR \cite{zhang2024feddcsr} 通过解耦表示学习实现了跨域联邦推荐。`

**问题5：注释不应出现在正文**
- **位置**：第122行
- **原文**：`（注：原"降低了许多"为口语化表述，学术写作建议替换为具体量化描述，若暂无数据可保留原表述）`
- **问题**：这是编辑注释，不应出现在最终论文中
- **建议修改为**：删除该注释，直接使用"降低了一个数量级"的表述

### 1.2 第二章（chap2_related.tex）

**问题6：表达不够准确**
- **位置**：第31行
- **原文**：`根据链接概率来判断用户和物品真实存在关系的可能`
- **问题**：表达不够准确和学术化
- **建议修改为**：`根据链接概率判断用户和物品之间是否存在真实关系` 或 `根据链接概率评估用户和物品之间存在真实关系的可能性`

**问题7：重复用词**
- **位置**：第356行
- **原文**：`空间复杂度主要主要涉及模型参数、图结构数据以及中间嵌入表示的存储。`
- **问题**："主要"重复
- **建议修改为**：`空间复杂度主要涉及模型参数、图结构数据以及中间嵌入表示的存储。`

### 1.3 第三章（chap3_method.tex）

**问题8：多余空格**
- **位置**：第20行
- **原文**：`基于 原型的语义对齐`
- **问题**："基于"和"原型"之间有多余空格
- **建议修改为**：`基于原型的语义对齐`

**问题9：重复用词**
- **位置**：第356行（与第二章相同问题）
- **原文**：`空间复杂度主要主要涉及模型参数、图结构数据以及中间嵌入表示的存储。`
- **问题**："主要"重复
- **建议修改为**：`空间复杂度主要涉及模型参数、图结构数据以及中间嵌入表示的存储。`

**问题10：图表注释格式**
- **位置**：第499行
- **原文**：`\textbf{注：}不同 $\alpha$ 值下的训练过程曲线图待补充。`
- **问题**：这是临时注释，应删除或补充实际图表
- **建议**：如果图表已准备好，应插入实际图片；如果尚未准备好，应删除该注释

**问题11：图表注释格式**
- **位置**：第452行
- **原文**：`\footnotesize{\\ 注：$^*$ 表示最优结果，且相比次优基线具有显著性差异 (p < 0.05)。}`
- **问题**：注释格式可以更规范
- **建议修改为**：`\footnotesize{注：$^*$ 表示最优结果，且相比次优基线具有显著性差异（$p < 0.05$）。}`（注意：删除多余的反斜杠，使用中文括号）

### 1.4 第四章（chap4_compression.tex）

**问题12：格式不一致**
- **位置**：第52行
- **原文**：`\textbf{按位聚合（Coordinate-wise Aggregation）}策略`
- **问题**：根据之前的修改要求，英文注释应删除或改为加粗形式
- **建议修改为**：`\textbf{按位聚合}策略` 或 `按位聚合（\textbf{Coordinate-wise Aggregation}）策略`

**问题13：图表注释**
- **位置**：第170行、第183行
- **原文**：`\textbf{注：}消融实验图表待补充。` 和 `\textbf{注：}去噪效应曲线图待补充。`
- **问题**：临时注释应删除或补充实际图表
- **建议**：如果图表已准备好，应插入实际图片；如果尚未准备好，应删除这些注释

## 二、逻辑与表达问题

### 2.1 第一章

**问题14：句子过长**
- **位置**：第30行
- **原文**：整个item内容是一个超长句，包含多个分句
- **问题**：句子过长，影响可读性
- **建议**：适当拆分，使用分号或重新组织句子结构

**问题15：表达不够清晰**
- **位置**：第32行
- **原文**：`本地模型陷入"零样本"困境`
- **问题**："零样本"在此处使用可能不够准确，通常"零样本"指没有训练样本，但这里是指缺乏交互数据
- **建议修改为**：`本地模型因缺乏历史交互数据而难以有效学习` 或 `本地模型面临交互数据匮乏的困境`

### 2.2 第三章

**问题16：引用格式不一致**
- **位置**：第291行等算法描述部分
- **原文**：`(\text{Sec 3.3})`、`(\text{Eq. 3.8})`等
- **问题**：应使用LaTeX的`\ref{}`命令进行交叉引用，而不是手动文本
- **建议修改为**：`\ref{subsec:attr_sem_graph}`、`\eqref{eq:cl_loss}`等

## 三、标点符号问题

### 3.1 全角半角混用

**问题17：括号使用不一致**
- **位置**：多处
- **问题**：中文语境下应使用全角括号，但部分地方使用了半角括号
- **建议**：统一使用全角括号，例如：`（p < 0.05）` 应改为 `（$p < 0.05$）`

## 四、术语统一性问题

### 4.1 术语不一致

**问题18：术语混用**
- **位置**：全文
- **问题**：部分地方仍使用"SOTA"等英文术语，应统一为中文表述
- **建议**：全文统一术语，例如"SOTA"改为"最优"或"先进"

## 五、格式规范问题

### 5.1 图表引用

**问题19：图表引用不规范**
- **位置**：多处
- **问题**：部分图表引用使用了手动文本而非LaTeX交叉引用
- **建议**：统一使用`\ref{}`和`\eqref{}`命令
