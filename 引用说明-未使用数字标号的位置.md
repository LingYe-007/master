# 引用说明：未使用数字标号的位置

## 一、当前引用方式说明

- 全文均使用 **`\cite{key}`** 形式，biblatex 已配置为 **citestyle=numeric**，正文引用以**上标数字**（如 ¹ ² ³）显示。
- 若某处**未显示为数字**而显示为**字母数字 key**（如 `luo2024perfedrec`、`FedNCFluo2024perfedrec`），通常是因为：
  1. **该 key 在 `chapters/reference.bib` 中不存在**，biblatex 无法解析，会直接显示 key；
  2. 或该 key 存在但 **biber 未成功运行 / 未重新编译**，导致引用未解析。

## 四、建议处理方式（使全文仅显示数字标号）

1. **补全缺失条目**  
   在 **`chapters/reference.bib`** 中为上述 10 个 key 各增加一条完整条目（作者、标题、年份、出版信息等），保存后执行：
   - `biber output/main`
   - 再运行两遍 `xelatex -output-directory=output main.tex`
   这样这些位置会变为数字标号。

2. **或改为已有 key**  
   若某文献实际已在 bib 中以其他 key 存在，可将正文中的 `\cite{缺失key}` 改为 `\cite{已有key}`，并删除/不添加该缺失 key。

3. **确认编译流程**  
   保证每次修改 .bib 或引用后都执行：**xelatex → biber → xelatex → xelatex**，避免未解析引用显示为 key。
