# 目录相关设置位置说明

所有与「目录」相关的设置分布在下面几个文件中，按文件列出，方便你手动调整。

---

## 1. `main.tex`

### 目录深度（显示到第几级）
- **约第 81 行**：`\setcounter{tocdepth}{4}`
  - `0` = 只显示章（section）
  - `1` = 章 + 节（subsection）
  - `2` = 章 + 节 + 条（subsubsection）
  - `3` = 再加 paragraph
  - `4` = 当前：显示到第四级（段）

### 目录页的插入位置
- **约第 82 行**：`\input{TableofContents}`  
  前面可加 `\markboth{目\quad 录}{目\quad 录}` 等，用于页眉。

---

## 2. `TableofContents.tex`

- **第 1 行**：`\newpage`（目录前换页）
- **第 2 行**：`\markboth{目\quad 录}{目\quad 录}`（目录页页眉）
- **第 3 行**：`\tableofcontents`（真正生成目录）
- **第 4 行**：`\newpage`（目录后换页）

只改「目录这一页」前后是否换页、页眉写什么，就改这个文件。

---

## 3. `images/option.tex`（目录样式集中在这里）

### 3.1 目录标题（“目录”两个字）
- **约第 87 行**：`\renewcommand{\contentsname}{...}`
  - 当前：居中、三号黑体（15.75pt）、内容为「目   录」
  - 改标题文字、字号、是否居中，都在这里改。

### 3.2 摘要/ABSTRACT 在目录里的格式（chapter 级）
- **约第 101–117 行**：`\renewcommand*\l@chapter[2]{...}`
  - **第 105 行**：`\setlength\@tempdima{1.5em}%`  
    控制「摘要」「ABSTRACT」条目的**左缩进**（改数字即可调缩进）。
  - 这里还控制：是否加粗、点线、页码位置等（整段宏都在 101–117 行）。

### 3.3 正文目录各级格式（titletoc）

- **约第 149 行**：**第一级（章）**  
  `\titlecontents{section}[1.5em]{...}`  
  - `[1.5em]`：章标题（如「第一章 绪论」）的**左缩进**。  
  - `\fontsize{14pt}{16.8pt}\selectfont\heiti\bfseries`：四号黑体加粗。

- **约第 153 行**：**第二级（节）**  
  `\titlecontents{subsection}[1.5em]{...}`  
  - `[1.5em]`：节（如 1.1、1.2）的**左缩进**。  
  - `\contentslabel{2.5em}`：编号“1.1”占的宽度。  
  - 字体：小四号宋体。

- **约第 157 行**：**第三级（条）**  
  `\titlecontents{subsubsection}[2.8em]{...}`  
  - `[2.8em]`：条（如 1.2.1、4.4.1）的**左缩进**。  
  - `\contentslabel{3em}`：编号宽度。

- **约第 161 行**：**第四级（段）**  
  `\titlecontents{paragraph}[4em]{...}`  
  - `[4em]`：段的**左缩进**。  
  - `\contentslabel{3.5em}`：编号宽度。

每一行的 `\titlerule*[0.7pc]{$\cdot$}\contentspage` 是「点线 + 页码」的样式，可整体替换或微调。

### 3.4 图目录 / 表目录的标题
- **约第 89 行**：`\renewcommand{\listfigurename}{...}`（“图 目 录”）
- **约第 90 行**：`\renewcommand{\listtablename}{...}`（“表 目 录”）

### 3.5 图目录、表目录条目的格式
- **约第 165–166 行**：`\l@figure`、`\l@table`  
  控制图目录、表目录里每一行的缩进和样式（`\@dottedtocline{1}{0em}{2.3em}` 等）。

---

## 4. 哪些内容会进「目录」、以什么级别进

- **摘要**：`chapters/abstract.tex` 约第 21 行  
  `\addcontentsline{toc}{chapter}{摘\quad 要}`  
  → 用上面的 `\l@chapter` 格式。

- **ABSTRACT**：`chapters/abstract.tex` 约第 58 行  
  `\addcontentsline{toc}{chapter}{ABSTRACT}`  
  → 同样用 `\l@chapter`。

- **参考文献**：`chapters/Reference.tex` 约第 3 行  
  `\addcontentsline{toc}{section}{参考文献}`  
  → 用 `\titlecontents{section}` 的格式。

- **致谢**：`chapters/thank.tex` 约第 11 行  
  `\addcontentsline{toc}{section}{致谢}`  
  → 用 `\titlecontents{section}` 的格式。

- **各章（第一章、第二章…）**：由各章里的 `\section{第X章\quad ...}` 自动写入目录，格式由 `\titlecontents{section}` 控制。
