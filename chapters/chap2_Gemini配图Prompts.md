# 第二章 科研配图 — Gemini Pro 生成用 Prompts（中文 + 统一风格）

所有 prompt 均用**中文**撰写；每张图的描述末尾都附带**同一段风格说明**，保证整章插图风格一致。生成时请将「图内容描述」与「统一风格」一起复制给 Gemini。

## 统一风格说明（每张图都需带上）

生成每张图时，在具体内容描述后**务必加上**下面这段：

```
【统一插图风格】整张图采用学术论文插图规范：纯白或极浅灰（#f5f5f5）背景；主体用黑色（#333）线条与边框，次要标注用深灰（#666）；可选用单一主色（如深蓝 #2563eb）做强调，但不超过一种主色；无渐变、无阴影、无立体效果；所有文字为清晰无衬线字体；构图简洁、留白适中，适合学位论文印刷。
```

## 使用说明

1. **复制方式**：将某张图的整段 Prompt（含【统一插图风格】）复制到 Gemini 的输入框，一次性生成；风格段保证本章所有插图一致。
2. **统一风格**：每张图末尾的【统一插图风格】段落必须保留，这样多张图一起使用时视觉统一（白/浅灰底、黑/灰线、至多一种主色、无立体与渐变）。
3. **迭代**：若某张图不符合预期，可追加一句，例如：  
   - “请更偏示意图，减少装饰。”  
   - “只使用黑色和灰色，不要彩色。”  
   - “主要标注字体再大一点。”
4. **尺寸**：生成后可统一裁剪或缩放，建议宽高比 4:3 或 16:10，便于 LaTeX 中 `\includegraphics[width=0.8\textwidth]{...}` 使用。
5. **文字**：若图中个别文字识别不准，可在本地用绘图软件或 LaTeX TikZ 覆盖中文标注。

如需为某一张图单独加长或缩短描述，可以说明图号，我再按你的要求改一版。
